# 字符数量统计工具

写公众号文章时，需要控制字数...
写论文时，要求严格的字符限制...
做翻译工作，需要精确统计字数...
写小说时，想知道今天写了多少字...
今天，这些问题都有了完美的解决方案！

## 功能特点

* 一键清空文本
* 快速复制统计结果
* 内置示例文本
* 支持快捷键操作

## 使用说明

##### 检查只需3步：

- 打开工具网页
- 在文本框中输入或粘贴内容
- 统计结果实时显示

##### 高效技巧

- 使用 Ctrl+L 快速清空文本
- 使用 Ctrl+K 快速复制统计结果
- 点击"载入示例"体验完整功能
